type="password" → Ô nhập văn bản nhưng ký tự bị ẩn (•••••) → Không có validation riêng, nhưng thường kết hợp minlength, required → Dùng cho mật khẩu tài khoản
type="number" → Ô nhập số, có nút tăng/giảm (spinner) → Chỉ cho nhập số, có thể giới hạn bằng min, max → Dùng để nhập số lượng sản phẩm
type="tel" → Ô nhập số điện thoại → Không validate chặt, nhưng hỗ trợ bàn phím số trên mobile → Dùng cho nhập số điện thoại khách hàng
type="url" → Ô nhập link → Tự kiểm tra định dạng URL (http/https) → Dùng khi người bán nhập link sản phẩm hoặc website
type="date" → Bộ chọn ngày (calendar picker) → Tự kiểm tra định dạng ngày hợp lệ → Dùng chọn ngày giao hàng hoặc ngày sinh
type="radio" → Nút chọn tròn (chỉ chọn 1 trong nhiều lựa chọn) → Không validation riêng, nhưng thường dùng với required → Dùng chọn phương thức thanh toán (COD, chuyển khoản…)
type="checkbox" → Ô vuông tích chọn (chọn nhiều) → Không validation riêng → Dùng chọn nhiều sản phẩm hoặc đồng ý điều khoản
type="file" → Nút upload file → Có thể giới hạn loại file bằng accept → Dùng upload ảnh sản phẩm hoặc ảnh đại diện
type="range" → Thanh trượt (slider) → Giới hạn giá trị trong khoảng min–max → Dùng lọc giá sản phẩm (ví dụ từ 100k–1 triệu)
