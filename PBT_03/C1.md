Vì bạn không set box-sizing, nên mặc định là content-box → width chỉ tính phần content, chưa tính padding + border.

Sidebar:
width: 300px
padding: 20px × 2 = 40px
border: 1px × 2 = 2px
 Tổng thực tế = 300 + 40 + 2 = 342px

Content:
width: 660px
padding: 30px × 2 = 60px
border: 1px × 2 = 2px
Tổng thực tế = 660 + 60 + 2 = 722px

Tổng chiều rộng 2 khối:
342 + 722 = 1064px
Trong khi container chỉ có:960px
Tổng > container → không đủ chỗ → .content bị rớt xuống dòng
3. Cách sửa #1 (Dùng border-box) 
* {
    box-sizing: border-box;
}

.container {
    width: 960px;
    margin: 0 auto;
    border: 2px solid black;
}

.sidebar {
    width: 300px;
    padding: 20px;
    border: 1px solid #ccc;
    float: left;
    background: lightblue;
}

.content {
    width: 660px;
    padding: 30px;
    border: 1px solid #ccc;
    float: left;
    background: lightgreen;
}
4. Cách sửa #2 (KHÔNG dùng border-box) ❗

 Phải trừ padding + border khỏi width

Tính lại:
Sidebar:
Tổng mong muốn: 300px
padding + border = 42px
width mới = 300 - 42 = 258px
Content:
Tổng mong muốn: 660px
padding + border = 62px
width mới = 660 - 62 = 598px
