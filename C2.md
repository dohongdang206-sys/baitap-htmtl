1. "Sản phẩm A" (h2.title.highlight trong #featured)
===font-size
Các rule liên quan:

body { font-size: 16px }
.container { font-size: 14px } → kế thừa xuống
.card .title { font-size: 20px } → áp trực tiếp vào h2

 Rule .card .title ghi đè inheritance

 font-size = 20px

===color

Các rule:

body { color: #333 } (kế thừa)
.card { color: blue } (kế thừa)
#featured .title { color: red } (specificity cao)
.highlight { color: green !important } (có !important)

So sánh:

#featured .title (ID) mạnh hơn class bình thường
Nhưng .highlight có !important → thắng tất cả

 color = green
